Options des blancs :

Lors de l'agrégation des données de texte, les options suivantes sont disponibles :

- agrégation des nulls et des blancs : lorsque cette option est sélectionnée, les données nulles sont comptées comme champ de texte de longueur zéro. Cela signifie qu'ils sont traités comme des chaînes de caractères vides. Lorsque cette option n'est pas sélectionnée, les données nulles sont traitées comme d'autres données de texte.

- agrégation des blancs : lorsque cette option est sélectionnée, les textes blancs (par exemple " ") sont regroupés et considérés comme une chaîne de caractères vide. Lorsqu'elle n'est pas sélectionnée, les textes blancs sont traités comme d'autres données de texte.

Note : Dans Oracle, les chaînes de caractères vides et nulles sont les mêmes objets. Vous devez donc cocher ou décocher les cases afin d'obtenir des résultats cohérents.
