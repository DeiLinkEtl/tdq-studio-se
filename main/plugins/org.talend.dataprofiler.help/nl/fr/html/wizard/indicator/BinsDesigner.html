Création d'intervalles :

Dans ce panneau, vous devez définir les intervalles à utiliser lorsque vous souhaitez agréger des données continues en intervalles discrets.

Les intervalles créés ont tous la même taille, sauf le premier et le dernier.

La valeur minimale est le commencement du premier intervalle.

La valeur maximale est la fin du dernier intervalle.

Exemple: min = 0 max = 100 nb = 4
Les intervalles sont
[0, 25[ U [25, 50[ U [50, 75[ U [75, 100[
Lorsque le nombre d'intervalles est configuré à 0, aucun intervalle n'est créé : l'indicateur calcule la fréquence de chaque valeur de la colonne.
