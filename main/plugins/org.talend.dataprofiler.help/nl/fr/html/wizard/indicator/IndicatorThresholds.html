Seuils des indicateurs :

Vous pouvez définir ici les seuils attendus sur la valeurs des indicateurs.

L'intervalle utilisé ici est utilisé pour mesurer la qualité des données.

Si la valeur de l'indicateur est en dehors de l'intervalle défini, cela signifie que les données sont de mauvaise qualité.

Par exemple, si vous définissez un intervalle [0, 3] sur l'indicateur de blancs, cela signifie que le nombre de champs blancs doit être inférieur ou égal à trois.

Vous pouvez définir un seuil ou pas de seuil du tout. Vous pouvez configurer ces seuils par valeur ou par pourcentage par rapport au nombre de lignes.
