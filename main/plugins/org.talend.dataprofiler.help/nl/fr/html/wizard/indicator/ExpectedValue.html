Valeur attendue :

Vous pouvez définir la valeur attendue pour qu'elle soit la plus fréquente.

Lorsque le mode courant (valeur la plus fréquente) est différent de ce qui est attendu, un avertissement sera affiché dans l'onglet "Résultats d'analyse" de l'éditeur.

Lorsque la case "Ignorer la casse" est cochée, la comparaison entre la valeur courante et la valeur attendue est sensible à la casse.
