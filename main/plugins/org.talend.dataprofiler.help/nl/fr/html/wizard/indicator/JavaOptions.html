Options Java :

Lorsque Java est utilisé pour calculer cet indicateur, vous pouvez spécifier le mapping des caractères. Le premier champ doit contenir la liste des caractères à remplacer. Le second champ doit contenir la liste des caractères de remplacement.

Chaque caractère du premier champ sera remplacé par le caractère ayant la même position dans le second champ.

Par exemple, avec les valeurs suivantes dans les champs,
"abc0123ABC,;.:"
"aaa9999AAApppp"

un "a", "b" or "c" sera remplacé par "a"
un "0", "1", "2" or "3" sera remplacé par "9"
un ",", ";", ".", ":" sera remplacé par "p"
