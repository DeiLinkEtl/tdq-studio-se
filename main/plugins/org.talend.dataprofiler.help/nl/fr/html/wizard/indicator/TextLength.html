Longueur de texte :

Lorsque vous calculez la longueur de données de texte, les options suivantes sont disponibles :

- nombre de nulls : lorsque cette option est sélectionnée, les données nulles sont comptées comme champ de texte de longueur zéro. Lorsqu'elle n'est pas sélectionnée, les données nulles ne sont pas prises en compte dans les résultats.

- nombre de blancs : lorsque cette option est sélectionnée, les textes blancs (par exemple " ") sont comptés comme des champs de texte de longueur zéro. Lorsqu'elle n'est pas sélectionnée, les textes blancs ne sont pas pris en compte dans les résultats.

Note : Dans Oracle, les chaînes de caractères vides et nulles sont les mêmes objets. Vous devez cocher ou décocher les cases afin d'obtenir des résultats cohérents.
