Expressions SQL :

Dans la première page de l'assistant, définissez le nom du modèle de façon à le retrouver facilement.

Dans la deuxième page, définissez le modèle SQL à utiliser dans les analyses.
Le modèle doit être entouré de guillemets simples "'".
Une fois le modèle créé, vous pouvez le glisser et le déposer sur une colonne dans l'éditeur d'analyse de colonnes multiples.

Note :
Les modèles SQL sont différents des modèles réguliers.
Par exemple, si vous cherchez des chaînes de caractères commençant par 'S', le modèle régulier est 'S%', et non 'S.*'.
Les modèles SQL sont plus facilement supportés par les bases de données :
ils apparaissent dans la clause « LIKE » d'une instruction SQL.
Les requêtes utilisant des modèles SQL peuvent être plus rapides que les requêtes contenant des expressions régulières.
Mais elles sont généralement beaucoup moins puissantes.
Merci de vous référer à la documentation de votre Système de Gestion de Base de Données pour plus d'informations.
