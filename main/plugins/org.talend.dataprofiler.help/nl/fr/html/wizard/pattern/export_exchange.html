Exporter vers Talend Exchange :

Chaque modèle exporté est sauvegardé en tant que fichier zip dans le dossier de destination.

Créez un compte sur le serveur Talend Exchange sur http://www.talendforge.org/exchange si vous n'en possédez pas encore.
Chargez et partagez vos fichiers zip sur le serveur Talend Exchange.

Consultez http://www.talendforge.org/exchange/tos/help_user.php pour plus d'informations.
