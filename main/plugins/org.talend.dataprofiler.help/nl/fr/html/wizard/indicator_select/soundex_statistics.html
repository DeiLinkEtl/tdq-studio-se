Statistiques de fréquence Soundex

Les indicateurs de ce groupe utilisent l'algorithme Soundex construit sur le SGBD.

Ils indexent les enregistrements par sons. Ainsi, les enregistrements ayant la même prononciation (anglaise uniquement) sont codés et ont la même représentation afin de les faire correspondre malgré des différences mineures d'orthographe.

  • Table de fréquence Soundex : calcule le nomre d'enregistrements distincts les plus fréquents par rapport au nombre total d'enregistrements ayant la même prononciation.

  • Table de basse fréquence Soundex : calcule le nomre d'enregistrements distincts les moins fréquents par rapport au nombre total d'enregistrements ayant la même prononciation.
