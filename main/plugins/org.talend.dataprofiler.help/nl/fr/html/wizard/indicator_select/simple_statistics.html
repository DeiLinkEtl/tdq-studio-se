Statistiques simples

Ces indicateurs fournissent des statistiques simples sur le nombre d'enregistrements.

Ils permettent d'avoir une vue d'ensemble du contenue des colonnes, combien de données nulles y a-t-il, combien de données distinctes, etc.

  • Nombre de blancs : compte le nombre de lignes blanches. Un "blanc" est une donnée textuelle non nulle contenant uniquement un espace blanc. Notez qu'Oracle ne différencie pas les chaînes de caractères vides des valeurs nulls.

  • Nombre de valeurs par défaut : compte le nombre de valeurs par défaut.

  • Nombre de valeurs distinctes : compte le nombre de valeurs distinctes de votre colonne.

  • Nombre de doublons : compte le nombre de valeurs contenues plus d'une fois. Pensez à la relation suivante : Nombre de doublons + Nombre de valeurs uniques = Nombre des valeurs distinctes.

  • Nombre de nulls : compte le nombre de lignes nulls.

  • Nombre de lignes : compte le nombre de lignes.

  • Nombre de valeurs uniques : compte le nombre de valeurs distinctes ayant une seule occurrence. Le nombre est inférieur ou égal au nombre de valeurs distinctes.

Par exemple, a,a,a,a,b,b,c,d,e => 9 valeurs, 5 valeurs distinctes, 3 valeurs uniques, 2 valeurs en doublon.
