Statistiques avancées

Elles déterminent les valeurs les plus probables et fréquentes et construisent des tables de fréquence. Les statistiques avancées principales comprennent les valeurs suivantes :

  • Mode : calcule la valeur la plus probable. Pour des données numériques ou continues, vous pouvez configurer des intervalles dans les paramètres de cet indicateur. Il est différent de "moyenne" et de "médiane". Ce mode est utile pour gérer des attributs de catégories.

  • Table de fréquence : calcule le nombre de valeurs les plus fréquentes pour chaque enregistrement distinct. D'autres indicateurs de tables de fréquence sont disponibles pour agréger les données par "date", "semaine", "mois", "trimestre", "année" et "intervalle".

    Note : Les statistiques de tables de fréquence sont appliquées uniquement sur les colonnes ayant des données de type "date".

  • Table de basse fréquence : calcule le nombre des enregistrements les moins fréquents pour chaque enregistrement distinct. D'autres indicateurs de tables de basse fréquence sont disponibles pour chacune des valeurs suivantes : "date", "semaine", "mois", "trimestre", "année" et "intervalle" où "intervalle" agrège les données numériques par intervalle.
