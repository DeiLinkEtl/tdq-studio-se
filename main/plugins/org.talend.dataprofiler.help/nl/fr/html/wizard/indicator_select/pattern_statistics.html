Statistiques de fréquence des modèles

Les indicateurs de ce groupe déterminent les modèles les plus et les moins fréquents.

  • Table de fréquence des modèles : calcule le nombre d'enregistrements les plus fréquents pour chaque modèle distinct.

  • Table de basse fréquence des modèles : calcule le nombre d'enregistrements les moins fréquents pour chaque modèle distinct.

  • Table de fréquence des modèles de date : récupère les modèles de date de colonnes de date ou de texte. Fonctionne uniquement avec le moteur Java.
