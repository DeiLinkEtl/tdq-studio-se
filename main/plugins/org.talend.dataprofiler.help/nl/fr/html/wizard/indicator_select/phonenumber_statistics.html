Statistiques des numéros de téléphone

Les indicateurs de ce groupe compte les numéros de téléphone. Ils retournent le nombre de chaque format de numéro de téléphone.

Ils valident les formats des numéros de téléphone à l'aide de la bibliothèque org.talend.libraries.google.libphonumber.

  • Nombre de numéros de téléphone valides : calcule le nombre de numéros de téléphone valides.

  • Nombre possible de numéros de télépohne : calcule les numéros de téléphone supposés être valides.

  • Nombre de codes région valides : calcule les numéros de téléphone ayant un code région valide.

  • Nombre de codes région invalides : calcule les numéros ayant un code région invalide.

  • Nombre de numéros de téléphone nationaux bien formés : calcule les numéros de téléphone nationaux bien formatés.

  • Nombre de numéros de téléphone internationaux bien formés : calcule les numéros de téléphone internationaux respectant le format international des numéros de téléphone (commençant par le code pays).

  • Nombre de numéros de téléphone E164 bien formés : calcule les numéros de téléphone internationaux respectant le format international des numéros de téléphone (maximum quinze chiffres avec un préfixe +).

  • Diagramme de fréquence des formats : affiche les résultats du nombre de numéros de téléphone dans un diagramme circulaire divisés en secteurs.

Ces indicateurs sont disponibles uniquement lorsque vous sélectionnez le moteur Java dans les paramètres de l'analyse.
