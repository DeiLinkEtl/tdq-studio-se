Analyse structurelle :

Analyse d'ensemble de schéma

Cette analyse retourne une vue d'ensemble du contenu d'un schéma. Elle calcule le nombre de tables et le nombre de lignes par table pour chaque catalogue et/ou schéma.

Utilisez-la pour analyser un schéma Oracle (et des schémas de certains autres SGBD). Si vous souhaitez analyser une base de données MySQL, choisissez l'analyse "Analyse de catalogue".
