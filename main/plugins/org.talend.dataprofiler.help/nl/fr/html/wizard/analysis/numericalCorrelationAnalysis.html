Analyse de corrélation numérique

Cette analyse affiche les corrélations entre des données nominales et numériques dans un graphique en bulles. Pour chaque valeur nominale, la moyenne des données numériques est calculée et représentée sur l'axe vertical et les valeurs nominales son représentées sur l'axe horizontal.

L'un des objectifs de cette analyse est de détecter facilement des problèmes de qualité de données, en regardant les valeurs extrêmes dans un graphique.
