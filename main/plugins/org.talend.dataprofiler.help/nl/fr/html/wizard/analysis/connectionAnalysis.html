Analyse structurelle :

Analyse d'ensemble des connexions

Cette analyse retourne une vue d'ensemble du contenu de votre base de données. Elle calcule le nombre de tables et le nombre de lignes par table pour chaque catalogue et/ou schéma, etc. Elle compte également le nombre d'index, de clés primaires, etc.

Cela vous permet de voir d'un coup d’œil quelle base de données contient le plus de données, quelles tables sont vides, etc.
