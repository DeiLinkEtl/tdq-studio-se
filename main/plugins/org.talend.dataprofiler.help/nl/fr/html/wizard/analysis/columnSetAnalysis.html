Analyse d'un ensemble de colonnes

Cette analyse s'applique à un ensemble de colonnes (pouvant définir en enregistrement complet) et non sur chaque colonne séparément, comme cela est le cas pour l'analyse de colonnes. Les statistiques présentées dans les résultats de cette analyse sont le nombre de lignes, le nombre de valeurs distinctes, le nombre de valeurs uniques et le nombre de valeurs en doublon, qui s'appliquent aux enregistrements (valeurs d'un ensemble de colonnes). Avec le moteur Java, vous pouvez également appliquer des modèles sur chaque colonne. Les résultats d'analyse retournent le nombre d'enregistrements correspondant aux modèles dans la liste.
Cette analyse récupère autant de lignes qu'il y a de lignes distinctes, afin de calculer les statistiques. Il est donc recommandé de ne pas sélectionner de clé primaire dans ce type d'analyse.
