Analyse de similaires

Cette analyse vous permet d'évaluer la quantité de données en doublon au sein de vos données. Elle estime le nombre de groupes de données similaires dans une table ou un ensemble de colonnes. Vous pouvez également l'utiliser pour définir des règles de correspondance qui peuvent ensuite être partagés dans le référentiel.

Visionnez la vidéo concernant la création de règles de rapprochement avec l'éditeur de mise en correspondance sur Talend Help Center.
