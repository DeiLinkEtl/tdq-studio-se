Analyse structurelle :

Analyse d'ensemble de catalogue

Ce type d'analyse vous donne une vue d'ensemble du contenu d'un catalogue. Il calcule le nombre de tables et le nombre de lignes par table pour chaque catalogue et/ou schéma.

Notez qu'Oracle (et quelques autres SGBD) ne définit pas les catalogues mais les schémas. Si vous souhaitez analyser une base de données Oracle, choisissez une "Analyse de schéma" à la place.
