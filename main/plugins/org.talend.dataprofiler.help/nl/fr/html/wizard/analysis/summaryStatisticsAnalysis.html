Analyse de colonnes :

Analyse de résumé statistique

Cette analyse vous permet d'analyser des données numériques. Elle crée une analyse de colonnes dans laquelle les indicateurs de résumé statistique sont automatiquement ajoutés aux colonnes numériques. Cela vous permet d'avoir une bonne idée de la forme de vos données numériques en calculant l'écart, l'écart interquartile, la moyenne et les valeurs médianes.
