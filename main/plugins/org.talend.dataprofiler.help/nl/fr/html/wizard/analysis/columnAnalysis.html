Analyse de colonnes :

Les analyses de ce dossier vous permettent de profiler les données de colonnes.

Tous les indicateurs sont calculés par rapport à une seule colonne (notre éditeur permet à l’utilisateur le faire sur plusieurs colonnes, mais chaque colonne est analysée séparément et indépendamment). Pour faire une analogie avec un fichier Excel, je dirais que cette analyse s'exécute sur une colonne et analyse chaque cellule de cette colonne.

Vous pouvez commencer par une analyse de colonne vide dans laquelle vous pourrez définir manuellement tous les paramètres, ou vous pouvez utiliser les analyses prédéfinies utilisant un ensemble d'indicateurs choisis selon vos besoins.

Vous pouvez également utiliser l'assistant d'analyse d'exploration sémantique pour configurer automatiquement une analyse de colonne se basant sur des informations rassemblées dans le référentiel sémantique.
