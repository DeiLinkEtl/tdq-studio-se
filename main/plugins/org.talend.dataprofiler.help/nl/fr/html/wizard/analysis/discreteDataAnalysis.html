Analyse de colonnes :

Analyse discrète de données

Cette analyse vous permet d'analyser des données numériques. Elle crée une analyse de colonnes dans laquelle les indicateurs, appropriés pour les données numériques, sont configurés.

Cette analyse utilise l'indicateur de fréquence d'intervalle (Bin) que vous pouvez configurer plus avant, afin de discrétiser les données selon vos besoins. Pour ce faire, ouvrez les options de l'indicateur.
