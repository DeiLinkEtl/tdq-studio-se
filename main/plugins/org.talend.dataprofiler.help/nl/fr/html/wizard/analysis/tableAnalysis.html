Analyse de tables :

La granularité la plus fine de cette analyse est la ligne. Vous souhaitez savoir, par exemple, si plusieurs lignes sont identiques ou similaires. Elle regroupe toutes les analyses dans le dossier "Analyses de tables". Une ligne est un ensemble de cellules appartenant à toutes les colonnes. Dans ce type d'analyse, la ligne est l'élément indivisible que vous analysez. Le studio permet à l'utilisateur de choisir sur quelle colonne il souhaite travailler. Ici, effectuez une analyse de table "partielle", dans le sens où l'élément indivisible n'est pas une ligne complète, mais un ensemble de cellules (appartenant à différentes colonnes, mais pas à toutes les colonnes). Cette analyse vous permet d'appliquer vos règles de qualité de données (Règles DQ) sur une table sélectionnée.

Notez que vous pouvez créer des règles DQ et les assigner à l'analyse créée, lors d'une étape ultérieure de la préparation de l'analyse.
