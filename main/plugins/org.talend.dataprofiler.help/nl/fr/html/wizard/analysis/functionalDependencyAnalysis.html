Analyse de dépendance fonctionnelle

Cette analyse vous permet de détecter des anomalies dans vos dépendances de colonnes. Elle détermine à quel point les valeurs d'une colonne A déterminent les valeurs d'une colonne B. Cette relation s'écrit A->B.

Par exemple, la valeur de la colonne “City” (ville) détermine la valeur des colonnes “Country” (pays), sauf en de rares exceptions.
