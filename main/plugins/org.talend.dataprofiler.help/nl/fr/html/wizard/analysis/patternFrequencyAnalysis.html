Analyse de colonnes :

Analyse de fréquence des modèles

Cette analyse vous permet de profiler des données en se basant sur les colonnes. Un ensemble d'"indicateurs" prédéfinis est assigné à chaque colonne sélectionnée. L'objectif de cette analyse est la découverte de modèles dans vos données. Elle montre les modèles fréquents mais aussi les modèles rares, afin que vous puissiez identifier les problèmes de qualité de données plus facilement.
