Analyse de colonnes :

Analyse d'exploration sémantique

L'assistant configure automatiquement une analyse de colonnes avec les indicateurs appropriés, après avoir appris de vos données.

Prérequis : Le référentiel sémantique doit être en cours d'exécution et accessible afin de démarrer cet assistant.

L'assistant essaye de deviner la signification de chaque colonne et de trouver le concept associé dans le référentiel sémantique. Selon les concepts, les indicateurs applicables peuvent automatiquement être trouvés et ajoutés aux colonnes.

Vous pouvez aider l'exploration sémantique en configurant une signification pour chaque colonne mais ce n'est pas obligatoire. Dans les dernières étapes de l'assistant, vous pouvez décider d'enrichir le référentiel sémantique avec vos entrées (significations des colonnes, synonymes). Enrichir le référentiel sémantique permet d'apporter de nouvelles informations aux studios connectés au même référentiel sémantique.

Lorsque l'analyse est exécutée, plus d'informations sont ajoutées au référentiel sémantique, comme les seuils sur les indicateurs sélectionnés. Ces informations sont réutilisées lorsque vous exécutez (ou un autre utilisateur exécute) cet assistant sur le même type de données.
