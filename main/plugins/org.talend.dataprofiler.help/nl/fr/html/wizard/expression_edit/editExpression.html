Modèles d'expressions :

Modèles d'expression

Vous pouvez insérer l'un des modèles en glissant-déposant l'élément dans la feuille des modèles.

Ces mots-clés peuvent être utilisés ici :
<%=__WHERE_CLAUSE__%>, remplace la clause WHERE fournie par le filtre des données ;
<%=__TABLE_NAME__%>,remplace le nom de la table ;
<%=__COLUMN_NAMES__%>,remplace la (les) colonne(s) ;
<%=__GROUP_BY_ALIAS__%>, remplace la clause complète "group by".
