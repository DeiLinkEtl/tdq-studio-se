Nouvel indicateur

L'assistant créé un indicateur personnalisé.

Merci de sélectionner le langage et d'entrer le modèle SQL dans l'assistant. Ces mots-clés peuvent être utilisés ici :
<%=__WHERE_CLAUSE__%>, remplace la clause where fournie par le filtre sur les données ;
<%=__TABLE_NAME__%>, remplace le nom de la table ;
<%=__COLUMN_NAMES__%>, remplace la (les) colonne(s) ;
<%=__GROUP_BY_ALIAS__%>, remplace la clause entière "group by".

Lorsque vous terminez l'indicateur personnalisé, l'éditeur s'ouvre, la catégorie par défaut de l'indicateur personnalisé est "Compte personnalisé".
